| **`useState`** | React Hook | `const [state, setState] = useState(initialValue)` <br><br> **รับพารามิเตอร์:** ค่าเริ่มต้น (เช่น null, 0, ข้อความ) | ข้างใน Component (`.jsx`) | เก็บและอัปเดตสถานะของข้อมูล (State) เมื่อมีการอัปเดต จะสั่งให้หน้าจอเรนเดอร์ใหม่โดยอัตโนมัติ | JSX (เพื่อนำ State ไปแสดงผล) | ต้องสร้างตัวแปร `let` ปกติ และต้องเขียนโค้ดแก้ DOM เอง (เช่น `element.innerText = ...`) ทุกครั้งที่ค่าเปลี่ยน |
| **`useEffect`** | React Hook | `useEffect(() => { ... }, [dependencies])` <br><br> **รับพารามิเตอร์:** 1. ฟังก์ชันที่ให้ทำงาน, 2. Array เงื่อนไข `[]` | ข้างใน Component (`.jsx`) | สั่งให้ทำงาน (เช่น ดึงข้อมูล API) **หลังจาก**ที่หน้าจอวาดภาพ (Paint) เสร็จเรียบร้อยแล้ว | `useState` (มักเอาข้อมูลที่ Fetch มาได้ ไปอัปเดตลง State) | ต้องดักจับ Event การโหลดหน้าจอ (เช่น `window.onload`) หรือแทรกฟังก์ชันในแท็ก HTML ตรงๆ |
| **`useContext`** | React Hook | `const value = useContext(MyContext)` <br><br> **รับพารามิเตอร์:** Context Object | ข้างใน Component (`.jsx`) | ดึงข้อมูลหรือฟังก์ชันจากส่วนกลางมาใช้ข้าม Component ได้เลย โดยไม่ต้องส่งต่อข้อมูลลงมาทีละชั้นๆ (Prop drilling) | `createContext()` และ `<Context.Provider>` | ต้องส่งข้อมูลผ่าน Props ต่อทอดลงไปทีละชั้นจนกว่าจะถึงปลายทาง (ยุ่งยากเมื่อซ้อนหลายชั้น) |
| **`createContext`** | React API (Setup) | `const MyContext = createContext(initialValue)` <br><br> **รับพารามิเตอร์:** ค่าเริ่มต้น | นอก Component / ไฟล์ตั้งต้น / หรือแยกไฟล์สำหรับ Context | สร้าง "คลังข้อมูลส่วนกลาง" หรือบริบทขึ้นมา เพื่อเตรียมแชร์ให้ Component ย่อยดึงไปใช้ | `useContext` และ `<Context.Provider>` | ไม่มีวิธีตรงๆ นอกจากส่ง Props ลงไปทีละระดับ หรือการใช้ Global Variables (ซึ่งอันตราย) |
| **`createBrowserRouter`** | Router API (Setup) | `const router = createBrowserRouter([ {path, element} ])` <br><br> **รับพารามิเตอร์:** Array ของออบเจ็กต์ Route | ไฟล์ตั้งต้นสุดของแอป (เช่น `main.jsx` หรือ `App.jsx`) | ประกาศโครงสร้างเส้นทางหน้าเว็บทั้งหมดแบบรวมศูนย์ รองรับทั้ง Nested และ Dynamic Routing | `<RouterProvider />` (เอาตัวแปล `router` ยัดใส่ Provider) | ต้องแยกสร้างไฟล์ HTML หลายๆ ไฟล์ (เช่น `index.html`, `about.html`) หน้าจอจะกระตุกโหลดใหม่ทุกครั้งที่เปลี่ยนหน้า |
| **`useParams`** | Router Hook | `const { id } = useParams()` <br><br> **พารามิเตอร์:** ไม่รับ (ดึงค่าจาก URL) | ข้างใน Route Component (`.jsx`) | ดึงตัวแปรที่แฝงมากับ URL ยืดหยุ่น (Dynamic Parameters) เช่น การดึง ID ของสินค้าเพื่อไปค้นหาต่อ | Dynamic Route ที่ตั้งไว้ใน Router (เช่น `path: "products/:productId"`) | ต้องเขียนโค้ดตัดคำจาก `window.location.pathname` ด้วย Regex หรือใช้ `URLSearchParams` เอง |
| **`useNavigate`** | Router Hook | `const navigate = useNavigate();` <br>`navigate('/path')` <br><br> **รับพารามิเตอร์:** String เส้นทางที่ต้องการย้ายไป | ข้างใน Route Component (`.jsx`) | สั่งเปลี่ยนหน้าเว็บ (Navigation) ด้วยคำสั่งโปรแกรมมิ่ง (เช่น สั่งเปลี่ยนหน้าทันทีเมื่อผู้ใช้กด Submit ฟอร์มสำเร็จ) | Event Listeners เช่น `onClick`, `onSubmit` | ใช้แท็ก `<a>` หรือสั่งเปลี่ยนหน้าด้วยคำสั่ง `window.location.href = "/path"` ใน JavaScript |
